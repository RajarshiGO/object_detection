This web application can detect wheat tips present in the image (for more information see the about page). There are two ways you can interact with the application, you can either upload an image that contains one or more wheat tips or you can provide an URL to a similar image. On starting the web app, you will be greeted by an ulpoad form that asks the user to upload an image. Upload an image and click on "Process Image" to get the bounding boxes.
If you want to provide an image URL then you can use the link available in the prompt displayed below the upload form. Alternatively, you can use the "Input Methods" drop down menu in the top bar to select the suitable input method. Provide the URL in the form and click on "Submit" to display the bounding boxes.
